# Fitness Planner WebApp

A web application to manage your workouts and nutrition efficiently, built with **Blazor Server**. The app helps users track meals, set nutritional goals, and organize training plans all in one place.

## Features

- **Folder Management:** Organize your training plans into folders for easy navigation.
- **Training Plan Creation:** Create and manage detailed workout plans.
- **Daily Meal Tracking:** Log meals for each day and monitor your nutritional intake.
- **Nutritional Goal Setting:** Define daily nutritional targets (e.g., calories, macros) and track your progress.
- **Meal & Goal Matching:** Automatically compare your logged meals against your set nutritional goals.
- **Dashboard Overview:** A central dashboard shows an overview of today's meals and pinned training plans.
- **Pinned Training Plans:** Keep important training plans accessible at a glance.
- **Visual Nutritional Tracking:** Graphical representation of tracked nutritional values to monitor progress.

## Technology Stack

- **Framework:** Blazor Server
- **UI:** Bootstrap, BlazorBootstrap
- **Database:** Entity Framework Core with Pomelo MySQL provider
